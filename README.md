# Nodemailer App

- [x] Email verification service

## setup

### install

```bash
npm install
```

### Wanna start it up

```bash
npm start
```

> or

```bash
npm run dev
```

> Pls make sure you fill this out in a .env file

```bash
MAIL_USERNAME=youremail@gmail.com
MAIL_PASSWORD=youremail-app-password
```
